## Canonical vs. Preprint Versions

Each paper exists in two forms:
1. **Canonical short version** (paper_NN_*.md) — the reference for book structure
2. **Long version (preprint)** — additional context and derivations

**Rule:** At contradictions, the canonical short version prevails.

## Consolidation Rules

1. **Canonical short version** (paper_NN) is always the reference
2. **Long version** (preprint) is used for additional derivations
3. At contradictions, **short version prevails**
4. All formulas must match `03_FORMULAS/formula_compendium.md`
5. Deprecated formulas from older preprints are **NOT adopted**
